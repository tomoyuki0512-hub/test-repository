# 遺族年金・遺族厚生年金 完全ガイド

亡くなった人に生計を維持されていた遺族が受け取れる公的年金です。日本の遺族年金は「**遺族基礎年金**（国民年金）」と「**遺族厚生年金**（厚生年金）」の2階建てになっています。

> 金額はすべて **令和7年度（2025年度）** の額です。年金額は毎年度改定されるため、最新値は日本年金機構の公式サイトで確認してください。

---

## 1. 遺族年金の全体像

| 種類 | 加入していた制度 | 主な受給対象 |
| --- | --- | --- |
| 遺族基礎年金 | 国民年金（全員が対象） | **子のある配偶者** または **子** |
| 遺族厚生年金 | 厚生年金（会社員・公務員） | 配偶者・子・父母・孫・祖父母 |

会社員・公務員が亡くなった場合は、両方を**あわせて**受け取れることがあります（2階建て）。自営業者など国民年金のみの場合は遺族基礎年金だけです。

---

## 2. 遺族基礎年金

### 受給条件

亡くなった人が次のいずれかに該当すること。

- 国民年金の被保険者である間に死亡した
- 老齢基礎年金の受給資格期間（25年以上）を満たした人が死亡した
- 保険料納付要件：死亡日の前々月までの被保険者期間のうち、**保険料納付済＋免除期間が3分の2以上**（または直近1年間に未納がない特例）

そのうえで、**受け取れるのは「子のある配偶者」または「子」だけ**です。子がいない配偶者は遺族基礎年金を受け取れません。

> **「子」の定義**：18歳到達年度の末日（高校卒業相当）までの子。障害等級1・2級の場合は20歳未満まで。

### 金額（令和7年度）

| 区分 | 年額 |
| --- | --- |
| 基本額（67歳以下・昭和31年4月2日以後生まれ） | **831,700円** |
| 子の加算 第1子・第2子 | 各 **239,300円** |
| 子の加算 第3子以降 | 各 **79,800円** |

**計算例：** 配偶者＋子2人 → 831,700 + 239,300 + 239,300 = **1,310,300円／年**

---

## 3. 遺族厚生年金

### 受給条件（亡くなった人の要件）

次の①〜④のいずれかに該当すること。①〜③を「**短期要件**」、④を「**長期要件**」と呼びます。

1. 厚生年金の被保険者（在職中）に死亡した
2. 被保険者期間中に初診日がある病気・ケガで、退職後**初診日から5年以内**に死亡した
3. 1級・2級の障害厚生年金の受給権者が死亡した
4. 老齢厚生年金の受給権者、または受給資格期間（25年以上）を満たした人が死亡した

加えて、遺族基礎年金と同じ**保険料納付要件**（3分の2以上）が必要です（短期要件の場合）。

### 受け取れる遺族と優先順位

生計を維持されていた遺族のうち、**最も優先順位の高い人**だけが受給できます。

1. **配偶者・子**
2. 父母
3. 孫
4. 祖父母

> ⚠️ **夫・父母・祖父母は、死亡当時に55歳以上であることが条件**で、支給開始は60歳から。
> ただし**夫**は、遺族基礎年金を受給中の場合に限り、60歳前でも遺族厚生年金をあわせて受給できます。

### 金額

亡くなった人の**老齢厚生年金（報酬比例部分）の4分の3（75%）**が基本です。

- 短期要件の場合、被保険者期間が**300月（25年）未満でも300月とみなして**計算（最低保障）

#### 報酬比例部分のざっくり計算式

```
報酬比例部分 ≒ 平均標準報酬額 × 5.481/1000 × 加入月数
遺族厚生年金   = 報酬比例部分 × 3/4
```

**計算例：** 平均標準報酬額 35万円・加入300月の場合
- 報酬比例部分 ＝ 350,000 × 5.481 ÷ 1000 × 300 ≒ **約575,500円**
- 遺族厚生年金 ＝ 575,500 × 3/4 ≒ **約431,600円／年**

### 中高齢寡婦加算（妻のみ）

次のいずれかに当てはまる**妻**には、40歳から65歳になるまで加算されます。

- 夫の死亡時に40歳以上65歳未満で、生計を同じくする子がいない
- 遺族基礎年金を受けていた子のある妻が、子が18歳年度末を過ぎて遺族基礎年金を受けられなくなった

**金額（令和7年度）：年額 623,800円**（※この加算は2028年4月から25年かけて段階的に廃止予定）

### 30歳未満・子のない妻の特例

子のない30歳未満の妻が受給権を得た場合、**5年間で受給権が消滅**します（有期給付）。

## 4. ⭐ パターン別シミュレーション（男性・女性・子ども）

会社員（厚生年金加入、報酬比例の遺族厚生年金を**約43万円／年**と仮定）が亡くなったケースで比較します。金額は令和7年度の目安です。

### パターン① 夫が死亡 → 妻（40歳）＋子2人

| 内訳 | 年額 |
| --- | --- |
| 遺族基礎年金（基本＋子2人） | 1,310,300円 |
| 遺族厚生年金 | 約431,600円 |
| **合計** | **約174万円／年** |

子が全員18歳年度末を過ぎると遺族基礎年金は終了し、その後は妻に**中高齢寡婦加算（約62万円）**が付く（遺族厚生年金＋加算＝約105万円／年）。

### パターン② 妻が死亡 → 夫（40歳）＋子2人

| 内訳 | 年額 |
| --- | --- |
| 遺族基礎年金（基本＋子2人） | 1,310,300円 |
| 遺族厚生年金（※遺族基礎年金受給中のため夫も受給可） | 約431,600円 |
| **合計** | **約174万円／年** |

> ⚠️ **男女差の注意点**：子が18歳年度末を過ぎて遺族基礎年金が終わると、夫が55歳未満の場合は**遺族厚生年金も支給停止**（60歳まで）。妻のような中高齢寡婦加算もありません。

### パターン③ 夫が死亡 → 妻（45歳）・子なし

| 内訳 | 年額 |
| --- | --- |
| 遺族基礎年金 | なし（子がいないため） |
| 遺族厚生年金 | 約431,600円 |
| 中高齢寡婦加算 | 623,800円 |
| **合計** | **約105万円／年** |

### パターン④ 妻が死亡 → 夫（45歳）・子なし

| 内訳 | 年額 |
| --- | --- |
| 遺族基礎年金 | なし |
| 遺族厚生年金 | **なし**（夫が55歳未満のため受給権なし） |
| **合計** | **0円** |

> これが現行制度で**最も大きい男女差**です（→ 2025年改正で見直し。下記参照）。

### パターン⑤ 子だけが受け取る場合

両親がいない、または配偶者に受給権がない場合、**子**が受給します。

| 内訳 | 年額（子2人の例） |
| --- | --- |
| 遺族基礎年金（基本＋第2子加算） | 831,700 + 239,300 = 1,071,000円 |
| 遺族厚生年金 | 約431,600円 |
| **合計** | **約150万円／年** |

> 子が複数いる場合は合算額を**人数で等分**して各自に支給。子が18歳年度末（障害時20歳）に達すると終了。

### パターン⑥ 30歳未満・子のない妻

遺族厚生年金 約431,600円／年を**5年間のみ**受給し、その後は受給権消滅。

---

## 5. 【重要】2025年改正で遺族厚生年金はこう変わる（2028年4月施行）

長年指摘されてきた男女差を解消する大改正が成立しました。

| 項目 | 改正内容 |
| --- | --- |
| 男女差 | 60歳未満で配偶者を亡くした場合、**男女とも原則5年間の有期給付**に統一 |
| 施行時期 | **男性は2028年4月から即時**、女性は2028年4月から**20年かけて段階的**に移行 |
| 受給額 | 有期給付は従来の**約1.3倍に増額** |
| 収入制限 | 現行の「年収850万円未満」要件を**撤廃** |
| 中高齢寡婦加算 | 2028年4月から**25年かけて段階的に廃止** |
| 継続給付 | 5年経過後も、障害状態の人や収入が十分でない人は増額された年金を継続受給可能 |

> 18歳までの子がいる間に支給される**遺族基礎年金**は、今回の改正の対象外（従来どおり）。

## 6. ⭐ 生涯受給額シミュレーション（具体例・現行／改定後）

実際のモデルケースで、**遺族年金を生涯でいくら受け取れるか**を試算します。

### 計算の前提

- **報酬比例部分** ＝ 平均標準報酬額 × 5.481/1000 × 加入月数
- 在職中の死亡（短期要件）のため、加入月数は **300月（25年）みなし**（40歳・36歳の実加入月数は300月未満のため、この最低保障が有利に働く）
- **遺族厚生年金** ＝ 報酬比例部分 × 3/4
- **子は0歳と仮定** → 遺族基礎年金は子が18歳到達年度末まで＝**約18年間**支給
- 平均寿命：**女性88歳・男性81歳**で試算
- 残された配偶者は厚生年金加入歴がない（専業的）と仮定 → 65歳以降も遺族厚生年金が満額継続
- 金額は令和7年度。**本人の老齢基礎年金（65歳〜）は「死亡による給付」ではないため除外**

> ⚠️ 子の年齢で遺族基礎年金の期間（＝総額）が大きく変わります。子が10歳なら遺族基礎年金は約8年分に減ります。

| ① 夫死亡（報酬50万） | 妻36・子0歳・専業 | **約5,820万円** | 約2,329万円※ |
| ② 妻死亡（報酬28万） | 夫40・子0歳 | **約1,928万円** | 約2,152万円 |
| ③ 夫死亡（報酬50万） | 妻36（年収450万）・子4歳 | **約4,223万円** | 約1,900万円※ |
| ④ 夫死亡（報酬70万） | 妻36・専業・子なし | **約4,489万円** | 約561万円※（継続給付次第） |

> ※女性は2028年4月から20年かけて段階移行。移行期に受給開始する場合の実額は「現行」と「改定後」の中間で、現行寄りになります。男性（②）は2028年4月から即時適用。

- 改正の狙いは**男女差の解消**。これまで手厚かった女性（終身）は縮小、ゼロに近かった若い男性は受給可へ、と方向が逆になります。
- いずれも**遺族基礎年金（子のいる間）は不変**。差が出るのは遺族厚生年金部分です。
- 5年の有期給付終了後も、**障害状態や低収入の場合は「継続給付」**で受給が続く可能性があります。

> これらは前提（子の年齢・配偶者の就労収入・平均寿命88歳/81歳・300月みなし・自身の老齢厚生年金額の仮定）を置いた**概算**です。とくに **①子の年齢（遺族基礎年金の期間）②残された配偶者の収入（65歳以降の併給調整・継続給付）③死亡時の年齢（中高齢寡婦加算や受給権の有無）** で総額は大きく変わります。正確な額は年金事務所・ねんきんネットでご確認ください。

## 8. ⭐ 繰下げ・繰上げ受給と遺族年金 — 受給開始年齢の戦略

> 設定：夫＝平均標準報酬月額60万・40年加入／妻＝平均標準報酬月額35万・38年（22〜60歳）加入。金額は令和7年度。繰上げ▲0.4%/月、繰下げ＋0.7%/月。

### 結論：**夫の繰下げ（繰上げ）は、妻の遺族年金を1円も増やさない**

これが最重要ポイントです。**遺族厚生年金は「亡くなった人の報酬比例部分の“本来額”× 3/4」**で計算され、**繰下げによる増額も繰上げによる減額も反映されません。**

- 夫の報酬比例（本来）＝ 60万 × 5.481/1000 × 480月 ＝ **1,578,528円**
- 遺族厚生年金（総額）＝ 1,578,528 × 3/4 ＝ **1,183,896円**（夫が60/65/70/75歳のどれで受給開始しても同じ）

### Q1. 夫の受給開始年齢別｜夫の年金と、妻の遺族年金

| 夫の受給開始 | 増減率 | 夫の存命中の老齢年金（本人） | 夫死亡後の妻の遺族厚生年金 |
| --- | --- | --- | --- |
| 60歳（繰上げ） | ×0.76 | 約 1,831,773円 | **約 1,183,896円（変わらず）** |
| 65歳（本来） | ×1.00 | 約 2,410,228円 | **約 1,183,896円（変わらず）** |
| 70歳（繰下げ） | ×1.42 | 約 3,422,524円 | **約 1,183,896円（変わらず）** |
| 75歳（繰下げ） | ×1.84 | 約 4,434,820円 | **約 1,183,896円（変わらず）** |

> 夫がいくら繰り下げて自分の年金を増やしても、亡くなった後に妻へ渡る遺族厚生年金は**一定**です。繰下げのメリットは「夫が長生きする間」だけ享受できます。

### 妻が65歳以降に実際に受け取る額（夫死亡後）

65歳以降は「自分の老齢厚生年金は全額」「遺族厚生年金は差額のみ」というルールで調整されます。

| 項目 | 年額 |
| --- | --- |
| 妻 自分の老齢基礎年金 | 790,115円 |
| 妻 自分の老齢厚生年金（全額） | 874,768円 |
| 遺族厚生年金（差額支給：1,226,648 − 874,768） | 351,880円 |
| **合計** | **約 2,016,763円／年** |

> ※遺族厚生年金は「① 1,183,896円」と「② 1,183,896×2/3＋妻の老齢厚生×1/2＝1,226,648円」の高い方から、妻自身の老齢厚生年金を差し引いた額。この合計額も**夫の受給開始年齢に関係なく一定**です。

### Q2. どちらを繰り下げるべき？（妻に楽をさせたい）

#### 💰 財力に余裕がある（支給を遅らせられる）場合

| 誰の年金 | 繰下げの効果 | おすすめ度 |
| --- | --- | --- |
| **夫の年金** | 夫が長生きする間は増える。**でも妻の遺族年金は増えない**（死亡で増額分は消える） | △（夫自身の長生き保険としてはアリ） |
| **妻の老齢“基礎”年金** | **遺族厚生年金と全額併給**できる＝繰下げ増額がまるごと一生残る | ◎（最優先） |
| 妻の老齢“厚生”年金 | 遺族厚生年金との差額調整で増額の旨味が相殺されやすい | ○〜△ |

**👉 妻に楽をさせたい最善策＝「妻自身の老齢基礎年金を繰り下げる」。**
理由：遺族厚生年金は妻の老齢“厚生”年金とだけ調整され、**老齢“基礎”年金とは全額併給**されるため、繰下げ増額がそのまま生涯（夫の死後も）上乗せされます。

- 妻の老齢基礎を**70歳まで繰下げ**：831,700 → **約1,181,014円**（＋約34.9万円/年）
- 妻の老齢基礎を**75歳まで繰下げ**：831,700 → **約1,530,328円**（＋約69.9万円/年）

夫は「自分が長生きするほど得」なので、夫婦の存命中の生活費に余裕があるなら夫も繰り下げてよいですが、それは**“妻の遺族年金対策”にはならない**点を理解しておくのが大切です。

> ⚠️ **重要な順番の注意**：妻が**遺族厚生年金の受給権を得る（＝夫の死亡）と、それ以降は妻自身の年金を繰り下げできません**（66歳前に遺族年金の受給権があると繰下げ不可、待機中に発生すると増額はその時点で固定）。つまり「妻の繰下げ作戦」は**夫がそれまで存命であること**が前提です。夫が早く亡くなりそうなら繰下げは成立しません。
> また、夫が老齢厚生年金を繰下げ待機している間は**加給年金（年下の妻がいる間の上乗せ）も受け取れません**。

#### 🪙 財力に余裕がない（支給を待てない）場合

- **無理に繰り下げる必要はありません。** 遺族年金は繰下げの有無と無関係なので、繰り下げなくても妻の遺族年金は減りません。
- 夫・妻とも**65歳から通常どおり受給**するのが基本。生活費を年金で賄う必要があるならこれが正解。
- **繰上げ（60歳開始）は非推奨**：減額（▲24%）が**一生続き**、しかも夫の繰上げでも遺族年金は減らないとはいえ、夫婦の存命中の総受給は目減りしやすい。どうしても生活費が足りないときの最終手段。
- できる範囲で、妻の老齢基礎年金だけでも数年繰り下げると、夫死亡後の妻の手取りが底上げされます（1ヶ月でも遅らせれば+0.7%）。

#### まとめ

1. **夫の繰下げ＝妻の遺族年金は増えない**。夫の繰下げは「夫の長生き保険」。
2. **妻に効くのは“妻自身”の繰下げ、とくに老齢基礎年金**（遺族厚生と全額併給）。
3. ただし**夫の死亡後は妻は繰下げできない**ので、繰下げ作戦は夫の存命が前提。
4. 財力がなければ無理せず65歳受給で十分（遺族年金は変わらない）。

> これらは前提を置いた概算です。実際は加入記録・夫婦の年齢差・加給年金・在職老齢年金などで変動します。正確な試算は年金事務所・ねんきんネットでご確認ください。
