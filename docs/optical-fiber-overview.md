# 光ファイバー・光通信 完全ガイド 総合インデックス

このドキュメント群は、**光ファイバーの仕組み・ネットワーク構成（FTTH/PON・海底ケーブル）・ケーブルと部材・施工と測定・主要メーカーと製品選定・伝送技術の最前線・業界とキャリア** を、**専門知識ゼロの人でも理解できる** ようにまとめたものです。

- 対象：光回線やネットワークの裏側を知りたい人、通信・DC業界の新人、光設備の設計・調達・施工に関わる実務者、業界研究中の学生
- 特徴：たとえ話 → 図解 → 規格・数値 → 実務（施工手順・製品選定）→ 最前線（コヒーレント・AIクラスタ）の順に、章が進むほど深くなります
- 進捗管理には **[🏠 学習ポータル](optical-fiber-portal.html)**（読了チェック付き）が便利です

> このページだけ読めば「全体像」と「どこに何が書いてあるか」がわかります。詳細は各章へ。

## 1. まず結論（30秒サマリ）

**光ファイバーとは「髪の毛ほどのガラスの糸に光を通して、データを速く・遠くまで・大量に運ぶ技術」** です。
光が糸から漏れないのは **全反射** のおかげで、その糸が **家（FTTH）→ 収容局 → 都市間幹線 → 海底ケーブル** と階層的につながって、インターネットの物理的な実体を作っています。

実務の全体像は、次の3行に集約されます：

| 何をする？ | キーワード | 章 |
| --- | --- | --- |
| **選ぶ** | ITU-T規格（G.652＝標準／G.654＝長距離／G.657＝曲げ耐性）とケーブル構造で決まる | ③⑤⑥ |
| **つなぐ** | 融着接続が品質の王様。コネクタは「青(UPC)と緑(APC)を混ぜない」 | ④ |
| **測る** | 損失はdBの足し算。合否はパワーメータ、場所の特定はOTDR | ④ |

そして鉄則がひとつ：**光障害の御三家は「汚れ・曲げ・接続不良」。接続前の清掃・検査が最強の予防策。**

## 4. <a id="用語集"></a>用語集（全章横断）

### 基礎（①）

| 用語 | 意味 |
| --- | --- |
| **コア** | 光が通る中心の芯。SMFで約9µm |
| **クラッド** | コアを包み光をはね返す層。外径125µm |
| **被覆／ジャケット** | 保護のための外装 |
| **全反射** | 光が屈折率の境界で100%はね返る現象。光ファイバーの基本原理 |
| **屈折率** | 光の曲がりやすさ。コア＞クラッドの差で全反射が起きる |
| **SMF（シングルモード）** | 細い芯で光が一直線。長距離・大容量向き |
| **MMF（マルチモード）** | 太い芯（50/62.5µm）。短距離・低コスト向き。OM3/4/5世代 |
| **減衰（損失）** | 進むうちに光が弱まること。約0.2dB/km（1550nm）。原因はレイリー散乱と吸収 |
| **波長** | 光の「色」。通信は1310nm（Oバンド）・1550nm（Cバンド）が主役 |
| **WDM／DWDM** | 波長の違う光を1本に多重して容量を増やす技術。幹線はDWDMで数十波 |

### ネットワーク（②）

| 用語 | 意味 |
| --- | --- |
| **FTTH** | Fiber To The Home。家まで光を引く方式 |
| **PON** | 1本の光をスプリッタで分けて最大32〜64世帯で共有する受動光網 |
| **OLT／ONU** | PONの局側親機／宅内子機（光⇄電気変換） |
| **光スプリッタ** | 電源不要で光を4〜64分岐する部品 |
| **GE-PON／10G-EPON／XGS-PON** | PONの規格。1ギガ／10ギガプランの中身 |
| **EDFA** | 光を光のまま一括増幅する光増幅器。DWDM幹線・海底中継器の心臓 |
| **陸揚局** | 海底ケーブルが陸に上がる拠点 |
| **ダークファイバ** | 敷設済みで未使用の心線を借りて自前機器で使う方式 |
| **レイテンシ** | 伝送遅延。ファイバ中の光速は約20万km/s（真空の2/3） |

### ケーブル・部材（③）

| 用語 | 意味 |
| --- | --- |
| **テープ心線** | 4/8/12心を横一列に貼り合わせた心線。多心一括融着できる |
| **テープスロット型** | 溝付きロッドにテープ心線を積層した超多心ケーブル構造 |
| **層より型** | テンションメンバの周りに心線を撚った中小心数向け構造 |
| **間欠リボン** | 間欠接着で柔軟性を持たせたテープ心線。数千心ケーブルの要 |
| **テンションメンバ（抗張力体）** | ケーブルの張力を受け持つ鋼線・FRP |
| **単心／メガネ／FOコード** | 1心／2心（8の字）／テープ心線を1心ずつ展開するコード |
| **ドロップケーブル** | 電柱から家への引き込み線 |
| **成端箱** | ケーブル端末で心線をコネクタ化して収める箱（屋内寄り） |
| **クロージャ** | ケーブル途中の接続・分岐点を防水保護する箱（屋外） |
| **ピグテール** | 片端だけコネクタ付きの短いコード。融着して成端に使う |

### 施工・測定（④）

| 用語 | 意味 |
| --- | --- |
| **融着接続** | アーク放電で溶かして一体化。損失0.02〜0.1dBの恒久接続 |
| **メカニカルスプライス** | V溝部品で機械固定する簡易接続。電源不要 |
| **現場組立コネクタ** | 現場でコネクタを直接取り付ける方式（e-SC・FuseConnect等） |
| **PC／UPC／APC** | フェルール研磨タイプ。UPC=青・APC=緑（8°斜め研磨）。**混在NG** |
| **dB／dBm** | 損失・利得の対数表現（-3dBで半分）／1mW基準の絶対パワー |
| **損失バジェット** | 送受信パワー差の範囲に経路損失を収める設計計算 |
| **OTDR** | パルス光の戻りで「どの距離で何dB損失か」を波形化する測定器 |
| **心線対照** | 曲げ漏れ光で使用中の心線かを無切断確認する保守技術 |
| **曲げ半径** | 許容できる曲げのきつさ。G.652=30mm、G.657=15〜5mm |

### 規格・製品（⑤⑥）

| 用語 | 意味 |
| --- | --- |
| **ITU-T G.652.D** | 世界標準のシングルモード規格。一般通信全般 |
| **G.654.E** | 超低損失・大有効断面積。幹線・海底・400G超向け |
| **G.657** | 曲げ耐性規格（A1/A2/B）。FTTH・宅内向け |
| **OM3/OM4/OM5** | マルチモードの世代。データセンター内 |
| **MFD（モードフィールド径）** | 光が実際に広がる径。接続互換性の指標 |
| **MPO** | 12〜24心一括の多心コネクタ。DC高密度配線の主役 |
| **コア調心／外径調心** | 融着機の調心方式。コア調心=低損失・高価、外径調心=経済的 |
| **Optigate／Pureシリーズ** | 住友電工の光製品ブランド／SMファイバ品種群 |

### 伝送・機器（⑦⑧）

| 用語 | 意味 |
| --- | --- |
| **波長分散／PMD** | 波長・偏波ごとの速度差でパルスが広がる現象。今はDSPで補償 |
| **非線形効果（カー効果）** | 強い光が屈折率を変え信号を歪ませる。パワーを上げられない理由 |
| **コヒーレント伝送** | 局発レーザーとの干渉で振幅・位相・偏波まで受信する方式。100G以降の幹線標準 |
| **QPSK／16QAM** | 位相・振幅の組み合わせで1シンボルに2/4ビット載せる変調方式 |
| **FEC（前方誤り訂正）** | 冗長ビットで誤りを数学的に訂正。pre-FEC BERが監視指標 |
| **シャノン限界** | 帯域とSNRで決まる容量の理論上限。C=B·log2(1+SNR) |
| **SDM／マルチコアファイバ** | 空間の多重で容量を増やす次世代技術。海底で商用化開始 |
| **ホローコアファイバ** | コアが中空のファイバ。光速約1.5倍で遅延3割減 |
| **QSFP-DD／OSFP** | 400G/800G世代のトランシーバの箱（フォームファクタ） |
| **SR／DR／FR／LR／ZR** | リーチ規格。MMF短距離〜コヒーレントDC間まで距離とファイバのセット |
| **PAM4** | 光の強さ4段階で1シンボル2ビット。DC内高速化の主役 |
| **シリコンフォトニクス／CPO** | 光回路の半導体集積／スイッチASICとの同一パッケージ化 |

### 資格・業界（⑨）

| 用語 | 意味 |
| --- | --- |
| **工事担任者** | 端末設備の接続工事を行う・監督する国家資格。光宅内工事の入口 |
| **電気通信主任技術者** | 事業用ネットワークの工事・維持・運用の監督責任者（伝送交換／線路） |
| **電気通信工事施工管理技士** | 通信工事の現場監督資格（1級/2級） |
| **通建（通信建設会社）** | キャリアの設計を現場に落とす施工会社。エクシオ・ミライト・コムシスが3大手 |
| **光ファイバセンシング** | 光ファイバで歪み・温度を測る非通信用途（DAS/DTS）。防災・インフラ監視で成長中 |
