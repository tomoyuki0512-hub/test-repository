# ケース面接 頻出お題集（練習問題バンク）

1人練習・模擬面接用のお題リストです。フェルミ推定25問＋ビジネスケース20問＋公共政策・抽象系5問の計50問を、難易度とヒント付きで収録しています。

> 解き方の型は [ケース面接 完全ガイド](case-interview-guide.md)、数値の暗記は [数値まとめ](case-interview-key-numbers.md) / [フラッシュカード](case-numbers-flashcards.html) を参照。

## 使い方

1. **1日1問**、お題だけ見て**ヒントを隠して**解く（フェルミ5分・ケース15〜20分）。ブラウザで練習するなら[練習トレーナー（HTML）](case-practice-trainer.html)が便利（タイマー・自己採点つき）。
2. 紙に「前提 → 構造 → 計算/分析 → 結論」を書き切ってから、ヒント列の分解式と見比べる。
3. 詰まった問題は翌週もう一度解く（2回目はヒントなしで通す）。
4. [3ヶ月スケジュール](case-interview-guide.md#10-対策スケジュール例3ヶ月ロードマップ)の2ヶ月目以降の教材として使う。★→★★→★★★の順に進めるとよい。

難易度の目安：★＝基礎（型どおりで解ける） / ★★＝標準（分解に一工夫） / ★★★＝応用（構造を自分で設計）

## 練習のチェック観点

解き終わったら毎回、この3つだけ自己採点する（[ガイド第8章の失敗リスト](case-interview-guide.md)も参照）。

- [ ] 前提・ゴールを確認してから構造化したか
- [ ] 分解がMECEで、どこがボトルネック/論点かを特定したか
- [ ] 結論ファーストで、根拠と数字をつけて言い切ったか
