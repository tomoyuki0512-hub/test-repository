# 年金と収入の関係（支給停止・収入要件まとめ）

「収入があると年金はもらえないの？」という疑問を、**老齢厚生年金（働きながら）** と **遺族厚生年金（収入要件）** に分けて整理します。結論を先に言うと——

- **老齢厚生年金**：働いて収入が高いと **一部/全部が止まる**（＝**在職老齢年金**）。ただし2026年4月から基準が大幅緩和。
- **遺族厚生年金**：**受給開始時の入口チェック（年収850万円未満）だけ**。いったん決まれば、その後収入が増えても**止まらない**。
- **老齢基礎年金・障害年金（原則）**：収入では止まらない。

> 金額は **令和8年度（2026年4月〜）** 基準。年金額・基準額は毎年度改定されます。

---

## 1. 老齢厚生年金 ×「働いて収入がある」→ 在職老齢年金

会社員として働きながら（厚生年金に加入しながら）老齢厚生年金を受け取ると、賃金と年金の合計が一定額を超えた分だけ、**老齢厚生年金が支給停止**になる仕組みです。これを **在職老齢年金** といいます。

### 仕組み・計算式

次の2つを足して、**支給停止調整額（基準額）** を超えた分の**半額**が止まります。

- **基本月額** ＝ 老齢厚生年金（報酬比例部分）の月額
- **総報酬月額相当額** ＝ その月の標準報酬月額 ＋ 直近1年の標準賞与額 ÷ 12

```
（基本月額 ＋ 総報酬月額相当額 − 基準額）÷ 2 ＝ 1か月の支給停止額
※ 結果がマイナス（合計が基準額以下）なら停止なし＝全額もらえる
```

### 基準額（支給停止調整額）の推移 — 2026年4月に大幅緩和

| 年度 | 基準額（月） | 備考 |
| --- | --- | --- |
| 令和6年度（2024） | 50万円 | |
| 令和7年度（2025） | 51万円 | |
| **令和8年度（2026年4月〜）** | **65万円** | 2025年改正で大幅引き上げ（法律上62万円＋賃金スライドで65万円）。**多くの人は減額されなくなった** |

### ⭐ 重要ポイント

- **老齢“基礎”年金は対象外**。収入がいくらあっても**全額もらえます**（止まるのは老齢“厚生”年金だけ）。
- **70歳以降も在職老齢年金は適用**されます（ただし厚生年金保険料の負担はなし）。
- 老齢厚生年金が**全額停止になると、加給年金も止まります**（一部停止なら加給年金は支給）。
- **個人事業主・フリーランスの収入は対象外**（厚生年金に入らない働き方なら、いくら稼いでも在職老齢年金では止まらない）。

### 計算例（老齢厚生年金 月13.2万円＝年約158万円の人）

| 働き方（総報酬月額相当額） | 令和7年度（基準51万）の停止額 | 令和8年度（基準65万）の停止額 |
| --- | --- | --- |
| 月40万円 | （13.2＋40−51）÷2＝**約1.1万円/月 停止** | 13.2＋40＝53.2万 < 65万 → **停止なし** |
| 月50万円 | （13.2＋50−51）÷2＝**約6.1万円/月 停止** | 13.2＋50＝63.2万 < 65万 → **停止なし** |
| 月60万円 | （13.2＋60−51）÷2＝**約11.1万円/月 停止** | （13.2＋60−65）÷2＝**約4.1万円/月 停止** |

> 2026年4月の改正で、**かなり高収入で働かない限り老齢厚生年金は止まらなく**なりました。

## 2. 遺族厚生年金 ×「収入」→ 入口の収入要件だけ

遺族厚生年金には在職老齢年金のような「働くと止まる」仕組みは**ありません**。関係するのは**受給を始めるときの入口チェック（生計維持要件）だけ**です。

### 生計維持要件（受給開始時の収入要件）

亡くなった人に生計を維持されていた遺族で、**年収850万円未満（または所得655.5万円未満）**であること。

- 判定は**死亡した時点**で行います。
- 死亡時に年収850万円以上でも、**おおむね5年以内に850万円未満になる見込み**（定年退職など）が書面で確認できれば認められます。

### ⭐ 重要ポイント

- **いったん受給が決まれば、その後に収入が増えても支給停止されません。** 遺族厚生年金を受けながら高収入で働いても止まりません（老齢年金の在職停止とは別物）。
- 逆に、入口で**不支給が確定すると、後で収入が下がっても受け取れません**（タイミングが重要）。
- **65歳以降の「差額支給」は“収入”ではなく“自分の老齢厚生年金”との調整**です（自分の老齢厚生年金を全額受け、遺族厚生年金は差額のみ）。詳しくは [遺族年金ガイドの併給調整](survivors-pension-guide.md#6--繰下げ繰上げ受給と遺族年金--受給開始年齢の戦略) を参照。
- **2025年改正後の「継続給付」**（5年有期給付の終了後）には**低収入要件**があり、就労収入が月約10万円（年約122万円）以下なら全額継続支給されます。

## 3. （参考）障害年金 × 収入

- **通常の障害年金（保険料を納めて受給）**：所得による支給停止は**ありません**。働いて収入があっても受け取れます（ただし障害状態の更新審査はあります）。
- **20歳前の傷病による障害基礎年金**（保険料納付なしで受給）：**所得制限あり**。本人の所得が一定を超えると半額または全額停止。

---

## 4. まとめ：どの年金が「収入」で止まる？

| 年金の種類 | 働いて収入が高いと止まる？ | 判定のしかた |
| --- | --- | --- |
| 老齢**基礎**年金 | ❌ 止まらない（全額） | — |
| 老齢**厚生**年金 | ⭕ 止まりうる（**在職老齢年金**） | 賃金＋年金が月**65万円**超で超過分の半額カット（令和8年度） |
| 遺族**基礎**年金 | ❌ 受給後は止まらない | 入口で生計維持要件（年収850万未満） |
| 遺族**厚生**年金 | ❌ 受給後は止まらない | 入口で生計維持要件（年収850万未満）。受給後の収入増は無関係 |
| 障害年金（原則） | ❌ 止まらない | — |
| 20歳前傷病の障害基礎年金 | ⭕ 所得制限あり | 本人の所得で半額/全額停止 |

### ひとことアドバイス
- **働きながら年金**：2026年4月の改正で在職老齢年金はかなり緩和。**月65万円（年金＋賃金）までは老齢厚生年金が満額**もらえます。気にせず働きやすくなりました。
- **遺族年金**：心配なのは**請求するタイミングの年収**だけ。受給が始まればバリバリ働いても止まりません。
