# 奨学金「代理返還（返還支援）制度」を実施している企業ガイド

> 入社すると、勤務先が奨学金（JASSO第一種・第二種）の返還残額の一部または全額を、本人に代わって**JASSOへ直接返してくれる**制度です。2021年4月開始。実質的に「返済しなくてよくなる（肩代わり）」福利厚生として、導入企業が急増しています。
>
> 👉 メインガイド: [大学進学のための奨学金・教育ローン 完全ガイド](daigaku-shogakukin-guide.md)（返還免除のセクション9と紐づく内容です）
>
> - 更新日: 2026年6月12日
> - **金額・条件は各社・各時点で変わります。応募前に必ず各企業の採用ページとJASSO公式検索で最新情報を確認してください。**

---

## まず押さえること

- **導入企業数は約4,800社**（2026年3月末時点）。製造・金融・小売・IT・建設・介護・士業など**業種は非常に幅広い**。
- 支援の形はさまざま:
  - 毎月の返還額と**同額**を会社が負担／**月◯万円まで**負担／**総額◯◯万円を上限**に負担、など。
  - **完済まで**支援する会社もあれば、**勤続◯年まで**等の期間を区切る会社もある。
- 本人にとって**受け取る返還相当額は所得税・住民税の課税対象外**（給与扱いにならない）＝手取りが減らない。
- 企業側も**損金算入**できる税制メリットがあり、人材確保策として広がっている。
- 出典: [JASSO 企業等の奨学金返還支援（代理返還）制度](https://www.jasso.go.jp/shogakukin/kigyoshien/index.html)

## ✅ 完全な「導入企業一覧」は公式検索で（約4,800社）

全社を載せたリストは存在せず、**JASSO公式の検索システムが唯一の網羅リスト**です。**地域・業種・企業名**で絞り込め、**検索結果はCSVでダウンロード**もできます。就活の企業選びでは、ここで志望業種・地元エリアを検索するのが確実です。

- 🔎 **学生向けポータル**: <https://dairihenkan-students.jasso.go.jp/>

## 就活でこの制度を活かすコツ

1. **JASSO公式検索**で、志望業種・地元エリアの導入企業をリストアップする。
2. 各社の**採用サイト／求人票の福利厚生欄**で「奨学金返還支援」「代理返還」の有無・上限・期間・対象（新卒のみか等）を確認する。
3. 面接・説明会で**「対象者の条件」「月いくら・総額いくらまで」「勤続要件」**を質問する。
4. **自治体の返還支援制度**（地方就職・移住で一部〜全額補助）と**併用できる**ケースもある。住む予定の自治体の制度も調べる（→ メインガイドのセクション9参照）。
