# 障害年金 完全ガイド

病気やケガで生活や仕事が制限されるようになったときに受け取れる公的年金です。現役世代でも受給でき、「**障害基礎年金**（国民年金）」と「**障害厚生年金**（厚生年金）」の2階建てです。

> 金額はすべて **令和7年度（2025年度）** の額（昭和31年4月2日以後生まれ）です。

---

## 1. 障害年金の全体像

| 種類 | 加入していた制度（初診日時点） | 等級 |
| --- | --- | --- |
| 障害基礎年金 | 国民年金（自営業・学生・専業主婦など） | 1級・2級 |
| 障害厚生年金 | 厚生年金（会社員・公務員） | 1級・2級・**3級**・障害手当金 |

会社員・公務員は、1・2級なら**障害厚生年金＋障害基礎年金の両方**を受け取れます。3級と障害手当金は厚生年金だけの上乗せです。

---

## 2. 受給するための3つの条件

### ① 初診日要件

障害の原因となった病気・ケガで**初めて医師の診療を受けた日（初診日）**に、国民年金または厚生年金に加入していること。**どちらに加入していたかで受け取れる年金の種類が決まります。**

### ② 保険料納付要件

初診日の前日時点で、初診日のある月の前々月までの被保険者期間のうち、

- **保険料納付済期間＋免除期間が3分の2以上**あること
- （特例）または直近1年間に保険料の未納がないこと

> 20歳前に初診日がある場合（先天性など）は保険料納付要件は問われず、20歳から「20歳前傷病による障害基礎年金」が受給できます（所得制限あり）。

### ③ 障害認定日要件

**障害認定日**（原則、初診日から**1年6か月**を経過した日。それより前に症状固定した場合はその日）に、法令の障害等級（1級・2級・3級）に該当する状態であること。

---

## 3. 障害等級の目安

| 等級 | 状態の目安 |
| --- | --- |
| **1級** | 他人の介助がなければ日常生活がほとんどできない |
| **2級** | 必ずしも介助は要さないが、日常生活が極めて困難で、労働で収入を得られない |
| **3級** | 日常生活はほぼ送れるが、労働に著しい制限がある（厚生年金のみ） |
| 障害手当金 | 3級より軽いが一定の障害が残った（一時金、厚生年金のみ） |

---

## 4. 金額（令和7年度）

### 障害基礎年金

| 等級 | 年額 | 月額換算 |
| --- | --- | --- |
| **1級** | **1,036,625円** | 約86,385円 |
| **2級** | **829,300円** | 約69,108円 |

**子の加算**（18歳年度末まで、または障害1・2級の20歳未満の子がいる場合）

| 区分 | 加算額 |
| --- | --- |
| 第1子・第2子 | 各 239,300円 |
| 第3子以降 | 各 79,800円 |

### 障害厚生年金

報酬比例部分（＝老齢厚生年金の計算に準じる）をベースに計算します。

| 等級 | 計算式 |
| --- | --- |
| **1級** | 報酬比例部分 × **1.25** ＋ 配偶者加給年金 |
| **2級** | 報酬比例部分 ＋ 配偶者加給年金 |
| **3級** | 報酬比例部分（**最低保障 623,800円**） |
| 障害手当金（一時金） | 報酬比例部分 × **2**（**最低保障 1,247,600円**） |

- **配偶者加給年金**：1・2級で、生計維持する**65歳未満の配偶者**がいる場合 **239,300円／年**
- 1・2級は、上記に**障害基礎年金（＋子の加算）が上乗せ**されます

### 障害年金生活者支援給付金（上乗せ）

| 等級 | 月額 |
| --- | --- |
| 1級 | 6,813円 |
| 2級 | 5,450円 |

## 5. ⭐ パターン別シミュレーション

報酬比例部分を**約60万円／年**（平均標準報酬額35万円・加入300月相当）と仮定した目安です。

### パターン① 自営業者（国民年金）が2級・子なし

| 内訳 | 年額 |
| --- | --- |
| 障害基礎年金 2級 | 829,300円 |
| **合計** | **約83万円／年** |

### パターン② 会社員（厚生年金）が2級・配偶者と子1人

| 内訳 | 年額 |
| --- | --- |
| 障害厚生年金 2級（報酬比例） | 約600,000円 |
| 配偶者加給年金 | 239,300円 |
| 障害基礎年金 2級 | 829,300円 |
| 子の加算 | 239,300円 |
| **合計** | **約191万円／年** |

### パターン③ 会社員（厚生年金）が1級・配偶者と子2人

| 内訳 | 年額 |
| --- | --- |
| 障害厚生年金 1級（報酬比例×1.25） | 約750,000円 |
| 配偶者加給年金 | 239,300円 |
| 障害基礎年金 1級 | 1,036,625円 |
| 子の加算（2人） | 478,600円 |
| **合計** | **約250万円／年** |

### パターン④ 会社員（厚生年金）が3級

| 内訳 | 年額 |
| --- | --- |
| 障害厚生年金 3級（最低保障） | 623,800円 |
| **合計** | **約62万円／年** |

> 3級は障害基礎年金がないため、厚生年金加入者だけの保障です。

### パターン⑤ 障害手当金（一時金）

3級より軽い障害が残って症状固定した場合、**一時金**として報酬比例部分の2倍（最低 1,247,600円）を一度だけ受け取ります。

---

## 6. 遺族年金との違い（早わかり）

| | 障害年金 | 遺族年金 |
| --- | --- | --- |
| 対象 | 病気・ケガで障害が残った**本人** | 亡くなった人の**遺族** |
| 3級・一時金 | あり（厚生年金） | なし |
| 男女差 | なし | あり（現行。2025年改正で見直し） |

---

## 7. 申請の窓口

- 障害基礎年金のみ：市区町村の年金窓口
- 障害厚生年金を含む：年金事務所・年金相談センター
- 障害認定日から請求できる（さかのぼり請求は最大5年分）
