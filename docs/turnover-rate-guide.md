# 離職率の適正値 完全ガイド

「離職率は何%が適正なのか？」という問いに、**計算方法・全国平均・業界別の目安・判断基準・改善策**の観点からまとめます。

> 結論から言うと、**「これが正解」という唯一の数字はありません**。全国平均は約15%前後で、業界や企業規模によって適正水準が大きく変わります。自社の値を平均・業界水準と比べ、**新卒3年以内離職率**や**離職の中身（誰が辞めているか）**まで見て判断するのが実務的です。

---

## 1. 離職率とは・計算方法

**離職率**＝一定期間に離職した人の割合。厚生労働省「雇用動向調査」では、次の式で年間の離職率を算出します。

```
離職率(%) = 年間の離職者数 ÷ 1月1日時点の常用労働者数 × 100
```

- **入職率**（新たに入った人の割合）と対で見るのが基本。
- 分母・分子の取り方（起算日・対象者・パート含むか）で数字が変わるため、**他社と比べるときは定義を揃える**こと。

### 派生指標（適正判断でより重要）

| 指標 | 意味 | なぜ重要か |
| --- | --- | --- |
| **新卒3年以内離職率** | 新卒入社が3年で辞める割合 | 定着・育成の実態を最もよく表す。就活・採用の判断材料 |
| **定着率** | 100% − 離職率（残った割合） | 求人票などでポジティブ表現に使われる |
| **早期離職率** | 入社1年以内など短期の離職 | ミスマッチ採用の兆候 |
| **自己都合／会社都合の別** | 辞めた理由の内訳 | 会社都合（リストラ等）が多いと別問題 |

---

## 2. 全国平均（ベンチマーク）

厚生労働省「雇用動向調査」による、日本全体のおおよその水準。

| 指標 | 目安の水準 |
| --- | --- |
| 全産業の離職率（年間） | **約15%前後** |
| 入職率 | 離職率と同程度（約15%前後）で推移 |
| 新卒(大卒)3年以内離職率 | **約3割（30〜35%）** ＝「七五三」の"三" |
| 新卒(高卒)3年以内離職率 | **約4割** |
| 新卒(中卒)3年以内離職率 | **約5〜6割** |

> **「七五三」現象**：新卒3年以内離職率が、中卒で約7割・高卒で約5割・大卒で約3割という、昔から知られた傾向を指す言葉。「大卒は3割が3年で辞める」が一つの基準線になります。

---

## 3. 業界別の傾向（適正水準は業界で変わる）

離職率は業界特性に強く依存します。**同じ10%でも、業界によって高いか低いかの評価が逆になる**ため、必ず同業と比較します。

### 離職率が高くなりやすい業界

| 業界 | 傾向 | 背景 |
| --- | --- | --- |
| 宿泊・飲食サービス業 | **最も高い（20〜30%台）** | 労働時間・シフト・パート比率が高い |
| 生活関連サービス・娯楽業 | 高い（20%前後） | 季節性・接客負荷 |
| 小売業 | やや高い | 店舗運営・パート中心 |
| 医療・福祉（介護など） | やや高い | 負荷の高さ・処遇 |

### 離職率が低くなりやすい業界

| 業界 | 傾向 | 背景 |
| --- | --- | --- |
| 製造業 | 低め（10%前後以下） | 長期雇用・技能蓄積 |
| 金融・保険業 | 低め | 安定志向・待遇 |
| 電気・ガス・水道（インフラ） | **最も低い部類** | 安定・公共性 |
| 情報通信業 | 中程度 | 転職市場が活発 |

> ポイント：**「業界平均より低ければ健全、高ければ要注意」** が第一の見方。全産業平均15%だけで判断しない。

---

## 4. 「適正値」の考え方

離職率には**低すぎても高すぎても問題**という側面があります。

### 高すぎる離職率のリスク

- 採用・教育コストの増大（1人採用に数十万〜百万円規模）
- ノウハウ・技術の流出、残る社員への負荷増
- 職場環境・マネジメントに問題があるサイン

### 低すぎる（≒ほぼ0%）ことの落とし穴

- 一見良いが、**新陳代謝が起きず組織が硬直**することも
- パフォーマンスの低い人材が滞留、変化に弱くなる
- 転職市場で戦えない＝外に出られないだけ、というケースも

### 目安としての適正レンジ

| 状態 | 年間離職率の目安 | 評価 |
| --- | --- | --- |
| 低すぎ | 〜3%未満 | 安定だが新陳代謝・多様性に注意 |
| **健全ゾーン** | **概ね 5〜15%** | 業界平均並みで大きな問題は起きにくい |
| やや高い | 15〜25% | 業界による。原因分析が必要 |
| 高い（要改善） | 25%以上 | 構造的な問題の可能性大 |

> ただしこれは**全産業の一般論**。飲食・小売なら20%台でも業界標準の範囲、製造・インフラなら10%でも高い、という具合に**業界補正が必須**です。

---

## 5. 自社が「適正か」を判断するチェックリスト

数字の高低だけでなく、**中身**を見ることが本質です。

- [ ] **業界平均と比べて**高いか低いか（全産業平均だけで判断しない）
- [ ] **新卒3年以内離職率**は業界水準・全国平均（大卒3割）と比べてどうか
- [ ] 辞めているのは**誰か**（優秀層／若手／特定部署に偏っていないか）
- [ ] **理由の内訳**（自己都合か会社都合か、待遇か人間関係か）
- [ ] **入職率とのバランス**（採れずに減り続けていないか）
- [ ] トレンド（前年比で悪化していないか）
- [ ] 会社規模の影響（一般に小規模ほど離職率は高めに出やすい）

> **「優秀な人・若手・同じ部署」から辞めているなら、率が平均並みでも危険信号**。逆に、率がやや高くてもポジティブな転職（キャリアアップ）中心なら問題は小さいこともあります。

---

## 6. 離職率を下げる（適正化する）主な施策

| 領域 | 施策例 |
| --- | --- |
| 採用 | 入社前後の期待値調整（RJP：現実を正直に伝える）、ミスマッチ防止 |
| オンボーディング | 入社後の育成・メンター制度、早期の不安解消 |
| 処遇 | 給与・評価制度の納得感、市場水準との比較 |
| 働き方 | 労働時間・残業の適正化、柔軟な勤務（リモート・時短） |
| 人間関係 | 1on1、心理的安全性、ハラスメント対策 |
| キャリア | 成長機会・キャリアパスの提示、社内異動の柔軟化 |
| 見える化 | 離職理由のヒアリング（退職面談）、サーベイで兆候を早期把握 |

> 施策は「率を下げること」自体が目的ではなく、**辞めてほしくない人が辞めない状態**を作るのがゴール。

---

## 7. よくある誤解

| 誤解 | 実際 |
| --- | --- |
| 「離職率は低いほど良い」 | 低すぎは新陳代謝の停滞リスク。適正ゾーンがある |
| 「全国平均15%を切れば安心」 | 業界による。飲食なら15%は低い、製造なら高いことも |
| 「離職率だけ見れば十分」 | 誰が・なぜ辞めたかの"中身"が本質 |
| 「求人票の定着率＝実態」 | 定義（対象者・期間）次第で数字は動く。定義を確認 |

---

## まとめ

- 離職率の「唯一の正解」はない。**全国平均は約15%前後**、大卒新卒3年以内は**約3割**が基準線。
- **健全ゾーンは概ね5〜15%**。ただし**業界補正が必須**（飲食・小売は高め、製造・インフラは低めが標準）。
- 低すぎ（〜3%）も新陳代謝の観点で注意。
- 率の高低より、**「誰が・なぜ辞めているか」**を見て、優秀層・若手の流出を防ぐことが適正化の本質。
