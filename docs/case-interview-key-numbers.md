# ケース面接で覚えておくべき数値まとめ

フェルミ推定・市場規模の概算では、**基礎となる数値を暗記しているかどうか**でスピードと精度が大きく変わります。ここでは覚えておくと武器になる各種の数値を分野別にまとめます。

> 数値は概数（覚えやすい丸め値）です。桁とオーダー感を掴むことが目的で、細かい正確さより「サッと出せること」を優先しています。
> ケースの解き方・対策方法は → [ケース面接 完全ガイド（内容と対策）](case-interview-guide.md)

## 11. 日本の代表企業の規模感（売上）

「兆円企業のオーダー感」を掴むためのアンカー。ケースで「大企業の売上を◯兆円と置く」ときの相場観に使う。

| 企業 | 年間売上（概数） | 業界 |
| --- | --- | --- |
| トヨタ自動車 | **約45兆円** | 自動車（日本最大） |
| 三菱商事 | **約20兆円** | 総合商社 |
| ホンダ | **約20兆円** | 自動車 |
| NTT | **約13兆円** | 通信 |
| ソニーグループ | **約13兆円** | エレクトロニクス・エンタメ |
| セブン&アイ | **約11兆円** | 小売（コンビニ） |
| 日立製作所 | **約10兆円** | 総合電機 |
| イオン | **約9.5兆円** | 小売（総合スーパー） |

- 目安：**日本のトップ企業で数十兆円、大手で数兆〜10兆円、上場大企業で数千億〜1兆円**のオーダー。
- 売上と利益は別物。製造・小売の営業利益率は数%〜10%程度、と併せて覚えておくと利益系ケースで使える。

---

## 12. 覚え方のコツ

- **「1.2億人」「100万人/学年」「600兆円GDP」** の3つは最優先で暗記。世界系は**「世界人口80億人」**をまず押さえる。
- 市場規模は「兆円オーダーか、千億オーダーか」の**桁だけ**まず押さえる。
- 数値は毎年少しずつ変わる。細かい端数より**オーダー（桁）と丸め値**を覚える。
- 面接では「〇〇を約△△と仮定します」と**根拠を宣言**すれば、多少ズレても減点されにくい。
