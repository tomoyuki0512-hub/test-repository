# Jekyllテーマ一覧

GitHub Pagesで追加設定なしに使えるテーマの一覧。

---

## テーマの変更方法

`docs/_config.yml` の `theme:` の行を書き換えるだけで変更できる。

```yaml
theme: jekyll-theme-cayman
```

変更後にコミット・プッシュすると数分で反映される。

---

## 使えるテーマ一覧

### jekyll-theme-cayman（現在使用中）
- 青緑のヘッダーが特徴
- シンプルで読みやすい
- ドキュメント向け

```yaml
theme: jekyll-theme-cayman
```

---

### minima
- GitHubの標準ブログテーマ
- ナビゲーションメニューあり
- 複数ページのサイト向け

```yaml
theme: minima
```

---

### jekyll-theme-minimal
- 余白が広くシンプル
- 左にサイドバーあり（タイトル・説明表示）
- 1ページのドキュメント向け

```yaml
theme: jekyll-theme-minimal
```

---

### jekyll-theme-primer
- GitHubのUIに近いデザイン
- 見慣れた印象でクセがない
- ドキュメント向け

```yaml
theme: jekyll-theme-primer
```

---

### jekyll-theme-slate
- ダークグレーのサイドバーが特徴
- コードが見やすい
- 技術系ドキュメント向け

```yaml
theme: jekyll-theme-slate
```

---

### jekyll-theme-hacker
- 黒背景・緑文字のターミナル風
- 個性的なデザイン
- 技術系・ポートフォリオ向け

```yaml
theme: jekyll-theme-hacker
```

---

### jekyll-theme-midnight
- 濃い青背景のダークテーマ
- 落ち着いた印象
- ポートフォリオ・プロジェクト紹介向け

```yaml
theme: jekyll-theme-midnight
```

---

### jekyll-theme-architect
- 青いヘッダーバーとサイドバー
- 構造的に整理されたレイアウト
- ドキュメント・仕様書向け

```yaml
theme: jekyll-theme-architect
```

---

### jekyll-theme-modernist
- グレー基調のミニマルデザイン
- すっきりした印象
- ポートフォリオ向け

```yaml
theme: jekyll-theme-modernist
```

---

### jekyll-theme-merlot
- ダークヘッダーとホワイトコンテンツ
- 落ち着いた2カラム風
- ブログ・ドキュメント向け

```yaml
theme: jekyll-theme-merlot
```

---

### jekyll-theme-dinky
- 左サイドバー固定のレイアウト
- コンパクトで軽量な印象
- 小規模サイト向け

```yaml
theme: jekyll-theme-dinky
```

---

### jekyll-theme-leap-day
- カラフルな左サイドバー
- 個性的なデザイン
- プロジェクト紹介向け

```yaml
theme: jekyll-theme-leap-day
```

---

### jekyll-theme-tactile
- 質感のあるテクスチャ背景
- 手作り感のあるデザイン
- クリエイティブ系向け

```yaml
theme: jekyll-theme-tactile
```

---

### jekyll-theme-time-machine
- クラシックなモノクロデザイン
- レトロな印象
- ドキュメント向け

```yaml
theme: jekyll-theme-time-machine
```
