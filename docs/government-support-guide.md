# 国からもらえる保障 まとめ

日本では、病気・ケガ・失業・出産・子育て・障害・老後・生活困窮など、人生のさまざまな場面で公的な保障（社会保障制度）が用意されています。**申請しないともらえない**ものが多いので、自分が対象になる制度を知っておくことが大切です。

> 金額の多くは **令和7年度（2025年度）** 時点の目安です。所得制限や地域差があるものは各窓口でご確認ください。
>
> 年金（老齢・障害・遺族）の詳細は別ガイドにまとめています。
> → [遺族年金・遺族厚生年金ガイド](survivors-pension-guide.md) ／ [障害年金ガイド](disability-pension-guide.md)

## 1. 医療・健康に関する保障

| 制度 | 内容 | 窓口 |
| --- | --- | --- |
| **高額療養費制度** | 1か月の医療費の自己負担が上限額を超えたら超過分が払い戻し。事前に「限度額適用認定証」で窓口負担を上限まで抑えられる | 加入する健康保険 |
| **傷病手当金** | 病気・ケガで連続3日休んだ後、4日目から最長1年6か月、**給与の約2/3**を支給（健康保険の被保険者＝主に会社員） | 加入する健康保険 |
| **出産育児一時金** | 出産で子1人につき **50万円** | 加入する健康保険 |
| **出産手当金** | 産前42日・産後56日の休業中、**給与の約2/3**（会社員） | 加入する健康保険 |
| **医療費控除** | 年間の医療費が10万円（または所得5%）を超えた分を所得控除 | 確定申告（税務署） |
| **自立支援医療** | 精神通院・更生・育成医療の自己負担を軽減（原則1割） | 市区町村 |
| **難病医療費助成** | 指定難病の医療費自己負担を軽減 | 都道府県・保健所 |

## 2. 仕事・失業に関する保障

| 制度 | 内容 | 窓口 |
| --- | --- | --- |
| **失業給付（基本手当）** | 離職後の生活を支える手当。離職理由・年齢・加入期間で給付日数が決まる（雇用保険） | ハローワーク |
| **求職者支援制度** | 雇用保険を受けられない人向け。職業訓練＋月10万円の給付金 | ハローワーク |
| **教育訓練給付** | 指定講座の受講費の一部（20〜70%）を給付 | ハローワーク |
| **育児休業給付** | 育休中に休業前賃金の**67%（181日目以降50%）**を支給。2025年から「出生後休業支援給付」で実質手取り10割に近づける仕組みも | ハローワーク |
| **介護休業給付** | 家族の介護で休業した期間、賃金の**67%**（最大93日） | ハローワーク |
| **労災保険（療養・休業補償）** | 業務上・通勤中のケガや病気の治療費・休業補償。保険料は全額会社負担 | 労働基準監督署 |

## 3. 子育て・ひとり親に関する保障

| 制度 | 内容 | 窓口 |
| --- | --- | --- |
| **児童手当** | 中学卒業まで→**2024年10月拡充で高校生年代（18歳年度末）まで**支給。第3子以降は月3万円に増額、所得制限は撤廃 | 市区町村 |
| **児童扶養手当** | ひとり親家庭などへの手当。所得に応じ支給（全部支給で月額 約45,500円〜、令和7年度） | 市区町村 |
| **出産・子育て応援交付金** | 妊娠時・出産時に計10万円相当 | 市区町村 |
| **就学援助** | 学用品費・給食費など、義務教育の費用を補助 | 市区町村・学校 |
| **高等学校等就学支援金** | 高校授業料を実質無償化（所得要件あり） | 学校・都道府県 |

## 4. 障害・介護に関する保障

| 制度 | 内容 | 窓口 |
| --- | --- | --- |
| **障害年金** | 病気・ケガで障害が残ったときの年金 → [詳細ガイド](disability-pension-guide.md) | 年金事務所・市区町村 |
| **特別障害者手当** | 重度の障害で常時介護が必要な在宅の人へ（月 約28,840円、令和7年度） | 市区町村 |
| **障害者手帳による各種割引・減免** | 税の控除、公共料金・交通機関の割引など | 市区町村 |
| **障害福祉サービス** | ホームヘルプ・就労支援など（自立支援給付） | 市区町村 |
| **介護保険サービス** | 40歳以上が加入。要介護認定で1〜3割負担で介護サービスを利用 | 市区町村 |
| **高額介護サービス費** | 1か月の介護自己負担が上限を超えたら払い戻し | 市区町村 |

## 5. 老後・年金に関する保障

| 制度 | 内容 |
| --- | --- |
| **老齢基礎年金・老齢厚生年金** | 原則65歳から受給。繰下げで最大84%増額も可能 |
| **遺族年金** | 一家の働き手が亡くなったとき → [詳細ガイド](survivors-pension-guide.md) |
| **年金生活者支援給付金** | 住民税非課税など低所得の年金受給者への上乗せ給付 |

## 6. 生活が苦しいときの保障（セーフティネット）

| 制度 | 内容 | 窓口 |
| --- | --- | --- |
| **生活保護** | 収入が最低生活費に満たない世帯へ、不足分を支給（生活・住宅・医療・教育などの扶助） | 市区町村（福祉事務所） |
| **生活福祉資金貸付** | 低所得・高齢・障害世帯への低利・無利子の貸付 | 社会福祉協議会 |
| **住居確保給付金** | 離職などで住まいを失うおそれのある人へ家賃相当を支給 | 自立相談支援機関 |
| **生活困窮者自立支援** | 家計改善・就労準備などの相談支援 | 自立相談支援機関 |

---

## 7. もらい忘れを防ぐポイント

1. **ほとんどの制度は「申請主義」** … 自動ではもらえない。自分から申請が必要。
2. **時効に注意** … 年金は5年、給付金にも請求期限がある。
3. **まず相談先を知る** … 迷ったら市区町村の窓口・年金事務所・ハローワーク・社会福祉協議会へ。
4. **併給調整に注意** … 制度によっては同時に満額もらえない（例：傷病手当金と障害年金など）。
