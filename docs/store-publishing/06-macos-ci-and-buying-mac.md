# macOS CIランナー（GitHub Actions等）料金と Mac購入費 の比較

「Macを持たずにiOSのビルド・署名・提出をする」もう一つの主役が **macOS CIランナー**（クラウド上の自動ビルド環境）です。
このページでは **GitHub Actions を中心に各サービスの料金** を詳しくまとめ、最後に **物理的にMacを買う場合の最低費用** と比較します。

（→ [AWS EC2 Mac ガイド](05-aws-mac-signing.md) / [Apple 公開フロー](01-apple-app-store.md)）

> ⚠️ **料金は変動します。** 以下はおおよその目安です。実際の契約前に必ず各社の公式料金ページを確認してください（リンクは末尾）。

---

## 1. 結論：あなたはどれを選ぶべき？

| あなたの状況 | おすすめ | ざっくり費用感 |
|---|---|---|
| たまにビルド・提出するだけ | **GitHub Actions（無料枠中心）/ Xcode Cloud 無料枠** | 0円〜数百円/月 |
| Flutterで手軽にCIしたい | **Codemagic / Bitrise の無料〜低額枠** | 0円〜数千円/月 |
| 毎日大量にビルドする | **物理Mac購入**（Mac mini）or 常設クラウドMac | 初期 約8.5万円〜（買い切り） |
| AWS内で完結したい/常設基盤 | [AWS EC2 Mac](05-aws-mac-signing.md) | 24時間課金で割高 |

**ポイント:** 使用頻度が低いほどクラウドCIの無料枠が有利、頻度が高いほど **Mac mini を1台買う方が結局安い**、という関係です。

---

## 2. GitHub Actions（macOSランナー）の料金 ★詳細

GitHubが提供するCI。`runs-on: macos-latest` と書くだけでクラウドのMacでビルドできます。

### 2-1. 課金の基本ルール

1. **パブリック（公開）リポジトリは標準ランナーが無料**（macOSランナーも無料枠の対象。ただしオープンソース向けの公正利用範囲）
2. **プライベートリポジトリ**は、プランごとの **無料「included minutes（含まれる分数）」** が使え、超過分は従量課金
3. **macOSは「分数の消費が割増（マルチプライヤー）」**：同じ1分でも **Linuxの10倍** 消費する

### 2-2. 分数マルチプライヤー（ここが重要）

無料included minutesは「Linux換算」で付与され、OSごとに以下の倍率で消費します。

| OS | 消費倍率 |
|---|---|
| Linux | ×1 |
| Windows | ×2 |
| **macOS** | **×10** |

つまり **含まれる分数が2,000分でも、macOSなら実質200分**しか使えません。

### 2-3. プラン別の無料included minutes（プライベートリポジトリ）

| プラン | 月額 | 含まれる分数(Linux換算) | macOS換算（÷10） |
|---|---|---|---|
| Free | 0円 | 2,000分 | 約200分 |
| Pro | 約4ドル/月 | 3,000分 | 約300分 |
| Team | 約4ドル/ユーザー/月 | 3,000分 | 約300分 |
| Enterprise | 個別 | 50,000分 | 約5,000分 |

### 2-4. 超過時の従量課金（macOSランナー）

標準のmacOSランナーの単価はおおよそ次の通り（USD/分、**1分単位で切り上げ**）:

| ランナー | 目安単価 |
|---|---|
| macOS 標準（3コア相当） | **約 $0.08/分** |
| macOS 大型（12コア等） | 約 $0.12〜/分 |

**試算（iOSビルド1回 ≒ 15分と仮定）:**

```
1回 ≒ 15分 × $0.08 = $1.2（約180円）
月20回ビルド ≒ $24（約3,600円）
月100回ビルド ≒ $120（約18,000円）
```

→ **月に数回〜十数回の提出なら無料枠でほぼ収まる**。毎日何十回も回すなら割高になり、Mac購入が見えてきます。

### 2-5. セルフホストランナーという選択肢

**自前のMac（手元のMac mini等）をGitHub Actionsのランナーとして登録**すれば、**GitHubの分数課金は一切かからない**（マシン代・電気代のみ）。「Mac mini購入 × セルフホスト」が、頻繁にビルドする人の鉄板パターンです。

## 3. その他の主要 macOS CI サービス

### 3-1. Xcode Cloud（Apple公式）

Appleが提供。Xcode/App Store Connectと密連携し、署名・TestFlight配布が楽。

| 項目 | 内容 |
|---|---|
| 課金単位 | **コンピュート時間（hours）** |
| 無料/下位枠 | 月25時間程度の枠を含むプランから（時期により変動） |
| 上位 | 100時間/250時間/1,000時間 などの月額プラン |
| 目安 | 100時間プランで月 約15ドル前後〜（変動） |

→ **Appleエコシステムで完結したい・署名や配布を簡単にしたい人**に好適。

### 3-2. Codemagic（Flutter/モバイル特化）

| 項目 | 内容 |
|---|---|
| 無料枠 | 月 **500分（M系Macインスタンス）** 程度 |
| 超過 | 従量（分単位）またはチームプラン |
| 特徴 | **Flutterのビルド・署名・両ストア提出**がGUIで設定しやすい |

→ **Flutterアプリなら第一候補級**に手軽。

### 3-3. Bitrise

| 項目 | 内容 |
|---|---|
| 無料枠 | 個人/小規模向けの無料プランあり（ビルド時間・同時実行に制限） |
| 有料 | チーム規模・マシンスペックに応じた月額 |
| 特徴 | モバイルCIの定番。ワークフローGUIが充実 |

### 3-4. CircleCI

| 項目 | 内容 |
|---|---|
| 課金単位 | **クレジット制**。macOSリソースクラスはクレジット消費が大きい |
| 無料枠 | 月一定クレジット |
| 特徴 | 汎用CIとして強力。設定はYAML中心 |

### 3-5. 常設クラウドMac（参考）

[AWS EC2 Mac](05-aws-mac-signing.md)、MacStadium、Scaleway/MacincloudなどはMacを**まるごと貸す**形態。CIランナーというより「クラウド上のMac本体」。常設・自由度が高い反面、**最低課金時間やマシン月額**で割高になりがち。

## 4. ★物理的にMacを買う場合の最低費用

iOSビルド・署名に使うだけなら、**最も安いのは Mac mini（デスクトップ・キーボード/モニタ別）** です。

### 4-1. 新品（最低スペックでOK）

| 機種 | 最低構成の目安 | 価格目安（日本/米国） |
|---|---|---|
| **Mac mini（現行 M4・最安デスクトップ）** | M4 / 16GBメモリ / 256GB SSD | **約 ¥84,800 / 約 $599** |
| MacBook Air（最安ノート） | M系 / 16GB / 256GB | 約 ¥148,000 / 約 $999〜 |

→ **新品で一番安く始めるなら Mac mini の最安構成（約8.5万円／約599ドル）**。
ディスプレイ・キーボード・マウスは手持ちの流用や安価な汎用品でOK（合計1〜2万円程度足せば一式そろう）。

> **ストレージ注意:** Xcodeだけで数十GB、プロジェクトやシミュレータ分も食います。256GBでも運用可能ですが、こまめな掃除が必要。予算が許せば512GBが快適。

### 4-2. 中古・型落ちでさらに安く

| 選択肢 | 価格目安 | 備考 |
|---|---|---|
| **中古 Mac mini M1**（2020） | **約 ¥40,000〜60,000** | iOSビルド用途なら十分実用的。最安級 |
| 中古 Mac mini Intel | 約 ¥20,000〜40,000 | 最新Xcode/OSの対応年数に注意（サポート切れリスク） |
| 整備済製品（Apple認定中古） | 新品の1〜2割引 | 保証付きで安心 |

> **最低スペックの考え方:** Apple Silicon（M1以降）+ メモリ8GB以上 + SSD 256GB以上 + 「最新Xcodeが動くmacOSバージョンに対応」していれば、ビルド用途は十分。**中古M1 Mac miniなら4〜6万円程度**が現実的な最低ラインです。

### 4-3. 買い切り vs クラウドの損益分岐（ざっくり）

```
Mac mini 新品 ≒ 85,000円（買い切り、数年使える）
GitHub Actions 従量 ≒ 月100回ビルドで約18,000円/月

→ 毎月たくさんビルドするなら、4〜5ヶ月でMac購入額を超える計算。
   頻繁に使うなら「Mac mini購入＋セルフホストランナー」が長期的に最安。
```

## 5. 総まとめ（早見表）

| 手段 | 初期費用 | ランニング | 向き |
|---|---|---|---|
| GitHub Actions（無料枠） | 0円 | 0円〜（超過は$0.08/分・macOS×10消費） | たまの提出 |
| GitHub Actions（セルフホスト） | Mac代 | 電気代のみ | 頻繁・Mac所有者 |
| Xcode Cloud | 0円 | 無料枠〜月15ドル前後 | Apple完結・配布重視 |
| Codemagic / Bitrise | 0円 | 無料枠〜従量 | Flutter/モバイル特化 |
| [AWS EC2 Mac](05-aws-mac-signing.md) | 0円 | 24時間最低課金で割高 | 常設・AWS完結 |
| **Mac mini 新品** | **約8.5万円** | 電気代のみ | 頻繁に使う・確実 |
| **中古 Mac mini M1** | **約4〜6万円** | 電気代のみ | 最低コストで自前環境 |

**ひとことで:**
- **年に数回**なら → GitHub Actions / Xcode Cloud の無料枠（実質0円）
- **頻繁に使う**なら → **中古Mac mini（約4〜6万円）or 新品Mac mini（約8.5万円）を買う**のが結局いちばん安くて速い
