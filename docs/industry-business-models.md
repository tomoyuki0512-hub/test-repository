# 業界別ビジネスモデル・KPIガイド（ケース面接用）

ケース面接で頻出の8業界について、**収益構造（売上の式）・コスト構造・主要KPIの相場観・ケースで突かれる典型論点**をまとめます。お題の業界の「儲けの仕組み」を知っていると、構造化とボトルネック特定が一気に速くなります。

> 数値の相場観は [数値まとめ](case-interview-key-numbers.md) と整合させた概数です。解き方の型は [完全ガイド](case-interview-guide.md)、実戦練習は [お題集](case-interview-question-bank.md) / [練習トレーナー](case-practice-trainer.html) へ。

---

## 1. コンビニ

```
売上 = 店舗数 × 日販 × 365
日販 = 来客数 × 客単価
```

| 項目 | 相場観 |
| --- | --- |
| 店舗数（全国） | 約56,000店（商圏 約2,000人/店） |
| 日販 | **約50〜70万円**（最重要KPI） |
| 来客数 / 客単価 | 800〜1,000人/日 / 約700円 |
| コスト構造 | 原価 約7割、ロイヤリティ・人件費・廃棄ロス |

**典型論点**：市場飽和（出店余地の枯渇）→ 日販を上げるしかない構造／人手不足と24時間営業の採算／中食・PB・サービス（宅配受取等)による客単価向上。

---

## 2. 外食（カフェ・ファストフード）

```
店舗売上 = 席数 × 回転率 × 営業時間 × 客単価 ＋ テイクアウト・デリバリー
```

| 項目 | 相場観 |
| --- | --- |
| 市場規模 | 外食全体 約25兆円 |
| FLコスト | 食材(Food)＋人件費(Labor)で**売上の55〜60%**が健全ライン |
| 客単価 | カフェ 500〜1,000円 / ファストフード 700円前後 / 居酒屋 3,000円前後 |
| 損益分岐 | 家賃は売上の10%以内が目安 |

**典型論点**：ピーク時間の席数制約（→回転率・テイクアウトで外す）／アイドルタイムの稼働／FLコストの継続的な上昇。

---

## 3. 小売・アパレル

```
売上 = 店舗数 × 坪数 × 坪効率（坪あたり売上）
     = 客数 × 買上率 × 客単価（点数 × 単価）
```

| 項目 | 相場観 |
| --- | --- |
| 市場規模 | アパレル 約9兆円・EC化率は約2割超 |
| 粗利率 | アパレル 50〜60%（ただし**値引き・残品ロスで利益が溶ける**） |
| 重要KPI | 在庫回転率、プロパー消化率（定価で売れた割合）、坪効率 |

**典型論点**：在庫リスク（作りすぎ→値引き）が利益の最大の敵／EC・OMO対応／SPA（製造小売）モデルと従来型の違い。

---

## 4. 航空

```
売上 = 路線数 × 便数 × 座席数 × 搭乗率(ロードファクター) × 平均単価(イールド)
```

| 項目 | 相場観 |
| --- | --- |
| 搭乗率 | **70〜80%が採算ライン目安** |
| コスト構造 | **固定費型**：機材・整備・人件費が大半。燃料は変動費で価格変動リスク大 |
| 重要KPI | 搭乗率、イールド（旅客キロあたり単価）、機材稼働率 |

**典型論点**：固定費型ゆえ「1席でも多く埋める」価値が大きい（ダイナミックプライシング）／LCCとFSCのコスト構造の違い／不採算路線の見極め（撤退基準）。

---

## 5. ホテル・旅館

```
売上 = 部屋数 × 稼働率(OCC) × 平均客室単価(ADR)
RevPAR（販売可能客室あたり売上）= 稼働率 × ADR ← 業界の最重要KPI
```

| 項目 | 相場観 |
| --- | --- |
| 稼働率 | 都市ホテル 70〜80%が好調ライン。旅館は50%前後も多い |
| コスト構造 | 固定費型（建物・人件費）。稼働率が損益を直撃 |
| 重要KPI | RevPAR、ADR、平日/休日・季節の稼働率格差 |

**典型論点**：閑散期・平日の稼働をどう埋めるか（法人・インバウンド・用途転換）／料飲・宴会など客室以外の収益源／価格を下げず稼働を上げる工夫。

---

## 6. SaaS・サブスクリプション

```
売上（MRR）= 顧客数 × 顧客単価(ARPU)
顧客数の増減 = 新規獲得 − 解約（チャーン）
LTV = ARPU × 粗利率 ÷ 月次解約率
```

| 項目 | 相場観 |
| --- | --- |
| 月次解約率 | BtoB SaaSで**1%未満が優良**、BtoC サブスクは数%が普通 |
| LTV/CAC | **3倍以上が健全**の目安（CAC＝顧客獲得コスト） |
| 粗利率 | 70〜80%と高いが、先行投資（開発・営業）で赤字先行しがち |

**典型論点**：解約率の改善は獲得より効く（分母がストック）／無料→有料の転換ファネル／プライシング（機能別・席数課金）。フィットネスやスマホゲームも同じ「ストック＋チャーン」構造で解ける。

---

## 7. フィットネスジム

```
会員数（ストック）= 期初会員 + 新規入会 − 退会
売上 = 会員数 × 月会費 × 12
```

| 項目 | 相場観 |
| --- | --- |
| 月会費 | 総合型 8,000〜12,000円 / 24時間型 7,000〜8,000円 |
| 月次退会率 | **2〜4%**（年間で会員の3割前後が入れ替わる） |
| コスト構造 | 固定費型（家賃・機材・人件費）。会員が増えても追加コストが小さい |
| 特徴 | 幽霊会員（来ないが払う会員）が収益を支える構造的皮肉 |

**典型論点**：入会後6ヶ月以内の定着が退会率のカギ／稼働の時間帯格差／24時間無人型との競争。→ [模擬面接スクリプト](case-interview-mock-script.md)がこの業界の実演。

---

## 8. 広告・メディア

```
広告売上 = 利用者数(リーチ) × 利用時間 × 広告単価
ネット広告 = 表示回数(imp) × クリック率(CTR) × クリック単価(CPC) など
```

| 項目 | 相場観 |
| --- | --- |
| 市場規模 | 総広告費 約7兆円、**うちネット広告 約3.3兆円**（TVを逆転済み） |
| 収益モデル | 広告（無料メディア）/ 課金・サブスク / ハイブリッド |
| 重要KPI | MAU/DAU、滞在時間、ARPU、広告単価（CPM/CPC） |

**典型論点**：「ユーザーの時間の奪い合い」構造（競合はテレビもSNSもゲームも）／広告依存か課金化か／メディアの売上はユーザー数×時間×単価に必ず分解できる。

## 横断で使える視点

| 視点 | 内容 |
| --- | --- |
| **固定費型 vs 変動費型** | 航空・ホテル・ジム・SaaSは固定費型 →「稼働率・会員数を1単位増やす」価値が大きく、値下げしてでも埋める打ち手が成立しやすい。小売・商社は変動費型 → 粗利率とロスの管理が本丸 |
| **フロー型 vs ストック型** | 外食・小売は毎日客を集めるフロー型。SaaS・ジム・通信はストック型 → **解約率**が最重要で、獲得よりリテンションが効きやすい |
| **稼働率ビジネスの共通式** | ホテル・航空・タクシー・パーキングはすべて「キャパ × 稼働率 × 単価」。ピークとオフピークの平準化が共通論点 |
| **利益率の相場** | 製造・小売の営業利益率は数%〜10%程度。SaaSの粗利は7〜8割。「利益率を◯%と置きます」の根拠に使う |
