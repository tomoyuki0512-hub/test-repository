# 投資指標と資産シミュレーション まとめ

リターン・標準偏差・シャープレシオ・相関係数の意味と計算方法、および一括投資・積立投資の成長シミュレーションをまとめたもの。

> 🧮 数値を自分で試したい場合は [投資指標・資産シミュレーター（HTML）](investment-simulator.html) を使うと、リターン・標準偏差・シャープレシオの計算と、複利・積立の成長グラフをブラウザ上で描画できます。

-----

## 第1部 4つの基本指標

### 1. リターン

**意味**：投資で得られる損益の割合。プラスなら利益、マイナスなら損失。資産の「期待される儲け」を表す。

**計算**

- 個別資産：`(期末価格 − 期初価格 + 配当) ÷ 期初価格`
- ポートフォリオの期待リターン ＝ 各資産のリターンを構成比で**加重平均**

```
E(Rp) = w₁·R₁ + w₂·R₂ + … + wₙ·Rₙ
```

**ポイント**：4指標のうち、リターンだけは単純な加重平均でそのまま求まる。

-----

### 2. 標準偏差（リスク）

**意味**：リターンのばらつきの大きさ＝**リスク**。大きいほど結果が上下に振れやすく、不確実性が高い。

**計算**

- 標本標準偏差（n−1 で割る不偏推定）

```
σ = √( Σ(xᵢ − x̄)² ÷ (n − 1) )
```

- 月次から年率への換算

```
σ年率 = σ月次 × √12     （日次なら ×√252）
```

- ポートフォリオの標準偏差は**加重平均にならない**。相関係数（共分散）が必要

```
σp = √( ΣᵢΣⱼ wᵢ·wⱼ·σᵢ·σⱼ·ρᵢⱼ )      （ρᵢᵢ = 1）
```

**ポイント**

- 算出には**月次リターン**を使う（年次だと点数が少なく不安定）。観測期間1/3/5/10/20年は「月次データの個数」が変わるだけ（12/36/60/120/240点）。
- 相関が低い資産を組むと、分散効果でリスクが加重平均より小さくなる。

-----

### 3. シャープレシオ

**意味**：取ったリスク1単位あたり、無リスク金利をどれだけ上回るリターンを得られたか。**リスク調整後リターン**の代表指標で、高いほど効率的な運用。

**計算**

```
シャープレシオ = (リターン − 無リスク金利) ÷ 標準偏差
```

**無リスク金利の決め方**

- 国債の利回りをそのまま代用する（計算式では求めない）。
- 運用期間に年限を合わせる：短期の評価 → 政策金利、長期・株式中心 → 10年国債利回り。
- 通貨を合わせる：円建てなら日本国債、ドル建てなら米国債。
- 参考（2026年6月時点の日本）：政策金利 約0.75%、10年国債利回り 約2.5〜2.7%。

**ポイント**：分散効果で標準偏差が下がると、シャープレシオは上がる。金利が上がると無リスク金利も上がり、同じ運用でもシャープは低めに出る。

-----

### 4. 相関係数

**意味**：2つの資産のリターンがどれだけ同じ方向に動くか。**−1（正反対）〜 0（無関係）〜 +1（完全に連動）**の範囲。

**計算**

```
ρ(x, y) = Cov(x, y) ÷ (σₓ × σᵧ)
        = 共分散 ÷ (両者の標準偏差の積)
```

**ポイント**

- 標本（n−1）でも母集団（n）でも、割り算が分子・分母で打ち消し合うため結果は同じ。
- 相関が低い資産を組み合わせるほど分散効果が大きい（リスク低減・シャープ改善）。
- N本のポートフォリオでは `N(N−1)/2` 個の相関が必要（例：6000本なら約1,800万個）。

-----

## 第2部 指標どうしの関係（分散投資の効果）

3ファンドのポートフォリオ例で、何が加重平均になり、何がならないかを比較する。

**前提**

| ファンド | 期待リターン | 標準偏差 | 構成比 |
|--------|------|----|---|
| A（国内株式） | 8% | 18% | 30% |
| B（先進国債券） | 3% | 6% | 40% |
| C（先進国株式） | 10% | 20% | 30% |

相関：ρ(A,B)=0.1、ρ(A,C)=0.7、ρ(B,C)=0.0／無リスク金利 Rf=1%

**結果**

| 指標 | 単純加重平均 | 正しい値 | 一致するか |
|-------|------|-----|----------|
| 期待リターン | 6.6% | 6.6% | ✅ する |
| 標準偏差 | 13.8% | 10.9% | ❌ しない（下がる） |
| シャープレシオ | 0.385 | 0.514 | ❌ しない（上がる） |

> 加重平均でそのまま使えるのは**期待リターンだけ**。標準偏差は相関を入れて計算し直す必要があり、相関が低いほどリスクが下がりシャープが改善する。これが分散投資の効果。

## 第3部 一括投資のシミュレーション（1万円）

**複利の式**

```
将来価値 = 元本 × (1 + 年率)^年数
```

**1万円が成長したときの評価額（円）**

| 経過年数 | 年3% | 年5% | 年7% | 年10% |
|----|------|------|------|-------|
| 1年 | 10,300 | 10,500 | 10,700 | 11,000 |
| 5年 | 11,593 | 12,763 | 14,026 | 16,105 |
| 10年 | 13,439 | 16,289 | 19,672 | 25,937 |
| 15年 | 15,580 | 20,789 | 27,590 | 41,772 |
| 20年 | 18,061 | 26,533 | 38,697 | 67,275 |
| 25年 | 20,938 | 33,864 | 54,274 | 108,347 |
| 30年 | 24,273 | 43,219 | 76,123 | 174,494 |

**成長倍率**

| 経過年数 | 年3% | 年5% | 年7% | 年10% |
|----|-----|-----|-----|------|
| 10年 | 1.34倍 | 1.63倍 | 1.97倍 | 2.59倍 |
| 20年 | 1.81倍 | 2.65倍 | 3.87倍 | 6.73倍 |
| 30年 | 2.43倍 | 4.32倍 | 7.61倍 | 17.45倍 |

> 利率の差は短期では小さいが、年数が伸びるほど複利で大きく開く。年10%・30年では元本の約17倍になる。

## 第4部 積立投資のシミュレーション（毎月 x 万円）

毎月一定額を積み立て、運用しながら増やしていくケース。投信のコスト（購入時手数料・信託報酬）を反映する。

**かかるコスト**

- **購入時手数料**：買うたびに申込額の数%がかかる手数料。積立では毎月の拠出ごとに差し引かれる。近年のインデックス投信・つみたてNISA対象は0%（ノーロード）が多い（別途、消費税がかかる）。
- **信託報酬（運用管理費用）**：保有している間ずっと、信託財産から日々差し引かれる年率コスト。インデックス型で年0.1〜0.2%、アクティブ型で年1〜2%程度。長期では複利で効くため最重要。
- （信託財産留保額：売却時にかかる場合がある0〜0.3%程度。今回は省略。）

**積立の式（手数料・信託報酬込み／毎月末に拠出・月複利）**

```
実投資額(月) = 月額 × (1 − 購入時手数料率)
実質リターン  = 市場リターン − 信託報酬率     （資産から日々差し引かれる）
  i = 実質リターン ÷ 12      （月利）
  N = 年数 × 12               （積立回数）

将来価値 = 実投資額 × ( (1 + i)^N − 1 ) ÷ i
元本     = 月額 × N           （手数料込みで自分が拠出した総額）
運用益   = 将来価値 − 元本
```

### 例1：手数料の有無・水準による違い（月3万円・市場リターン5%・30年）

| シナリオ | 実質リターン | 元本 | 評価額 | 運用益 | 手数料負担 |
|--------------------|------|----------|----------|----------|---------|
| 手数料なし（理論値） | 5.0% | 10,800,000 | 24,967,759 | 14,167,759 | — |
| 低コスト型（購入0%／信託0.2%） | 4.8% | 10,800,000 | 24,064,424 | 13,264,424 | 903,335 |
| 高コスト型（購入2.2%／信託1.5%） | 3.5% | 10,800,000 | 18,643,010 | 7,843,010 | 6,324,749 |

> 信託報酬は毎年資産から差し引かれて複利で効くため、年1.5%（＋購入時手数料）でも30年で約630万円の差になる。低コストのインデックスファンドが長期では大きく有利。

### 例2：低コスト型（購入0%／信託0.2%）月3万円の推移

| 経過年数 | 元本 | 評価額 | 運用益 |
|----|----------|----------|----------|
| 5年 | 1,800,000 | 2,029,805 | 229,805 |
| 10年 | 3,600,000 | 4,608,959 | 1,008,959 |
| 15年 | 5,400,000 | 7,886,136 | 2,486,136 |
| 20年 | 7,200,000 | 12,050,251 | 4,850,251 |
| 25年 | 9,000,000 | 17,341,345 | 8,341,345 |
| 30年 | 10,800,000 | 24,064,424 | 13,264,424 |

> 手数料なしの理論値（30年で24,967,759円）と比べ、信託報酬0.2%でも約90万円目減りする。

### 例3：信託報酬の水準別インパクト（月3万円・購入0%・市場5%・30年）

| 信託報酬（年） | 評価額 | 手数料なしとの差 |
|-------|----------|---------|
| 0.1% | 24,511,128 | 456,631 |
| 0.2% | 24,064,424 | 903,335 |
| 0.5% | 22,781,584 | 2,186,175 |
| 1.0% | 20,821,482 | 4,146,277 |
| 1.5% | 19,062,382 | 5,905,377 |
| 2.0% | 17,482,107 | 7,485,653 |

> 信託報酬が高いほど最終評価額は大きく削られる。同じリターンの商品なら信託報酬の低さがそのまま差になる。商品選びで最優先にチェックしたい項目。

## 付録：前提と注意

- **第4部は購入時手数料・信託報酬を反映済み**。ただし**税金は未考慮**（譲渡益・分配金に約20.315%課税。NISA等の非課税口座なら非課税）。第3部の一括投資は手数料等を含まない理論値。
- **リターンは一定と仮定**した単純試算。実際の相場は毎年変動し、暴落で評価額が大きく下がる年もある。
- 積立は**毎月末に拠出・年率÷12の月複利**を前提。計算方法（拠出タイミング、複利の取り方）が違うと結果は多少ずれる。
- 第1〜2部の数値も含め、**過去の実績や前提は将来を保証しない**。あくまで指標の意味とオーダー感をつかむためのもの。
